# Proyecto-Ingenieria-de-Software-
 La problemática se sitúa en el taller mecánico "El Güero Grupo Llantero", un negocio local que actualmente gestiona sus servicios y control de inventario de manera ineficiente debido a la dependencia de procesos manuales. 

El Sistema de Control- El Güero Grupo Llantero se desarrollara en una pagina web 
Herramientas:
Utilizaremos Xampp y phpMyAdmin para ejecutar la base de datos 
Utilizaremos Django para hacer la pagina web
